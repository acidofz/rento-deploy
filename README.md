# RENTO — Аренда жилья без риелторов

**RENTO** — это P2P-платформа для прямой аренды жилья в Узбекистане.  
Мы соединяем арендодателей и арендаторов напрямую, без посредников и переплат.

![Rento Banner](https://via.placeholder.com/1200x400/3b7eff/ffffff?text=RENTO)

## 🎯 О проекте

Каждый год в Узбекистане совершается более **340 000** арендных сделок. Большинство из них проходит через риелторов, которые берут комиссию 50–100% от месячной аренды.

**Rento** решает эту проблему:
- Арендодатель размещает объявление бесплатно
- Арендатор находит жильё напрямую
- Встроенный чат для общения
- Автоматическая генерация договора аренды
- Верификация пользователей для безопасности

## ✨ Основные возможности

### Для арендодателей
- Публикация объявлений с фото и подробным описанием
- Управление объявлениями (активация/деактивация)
- Чат с потенциальными арендаторами
- Генерация PDF-договора автоматически
- Продвижение объявлений (boost)

### Для арендаторов
- Умный поиск с фильтрами (район, цена, комнаты, этаж и др.)
- Просмотр объявлений на карте
- Чат с хозяином в реальном времени
- Сохранение понравившихся объявлений
- Просмотр аналитики цен по районам

### Общее
- Регистрация по номеру телефона + SMS
- Верификация личности (паспорт + селфи)
- Рейтинги и отзывы
- Push-уведомления
- Платежи через Click и Payme (в разработке)

## 🛠 Технологический стек

### Backend
- **Node.js** + **Fastify** (TypeScript)
- **Prisma ORM** + **PostgreSQL 16** + **PostGIS**
- **Redis** (кеш, сессии, очереди)
- **Socket.io** — real-time чат
- **BullMQ** — фоновые задачи

### Frontend
- **Next.js 14** (App Router) + TypeScript + Tailwind CSS
- **TanStack Query** + **Zustand**

### Mobile
- **React Native** + **Expo** (Android-first)

### Инфраструктура
- **Cloudflare R2** — хранение фотографий
- **Hetzner Cloud** (MVP)
- **Docker** + **GitHub Actions**
